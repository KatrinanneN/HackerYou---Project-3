Student at HackerYou

In 4 months, we tackled lessons that focused on HTML5 + CSS3 as well as fundamentals of acccessibility for each module. There were 3 projects that needed to be completed. The first 2 were designed by HackerYou. The design had to be replicated with our own code. The design and code of the last project had to be created by each of the students. Have a look.

PROJECT ONE

This project was a simple one. We had to reproduce the design using HTML5 + CSS3 languages with an emphasis on floats. It did not require responsive nor active links.

PROJECT TWO

This design had to be reproduced in HTML5 + CSS3 languages. At this point, we were tasked with using both the float and flexbox properties. It did not require responsive design nor any active links.

PROJECT THREE

This project was to be anything we wanted. I chose a portfolio/resume. It required responsive design and active links using HTML5 + SASS languages.
